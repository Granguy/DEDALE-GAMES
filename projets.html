FAQ

    • Escaryper est une application mobile sur le principe d’escape game, un jeu d’évasion et d’énigme dans le domaine de l'horreur.
    • Le jeu est partiellement prêt, une version alpha est disponible sur le site.
    • Issu d’une fusion des mots anglais “Escape” et “Scary” qui signifie “évasion” et “peur”, nous avons ajouté le suffixe “-per” signifiant les différentes professions en anglais.
    • Le jeu est entièrement gratuit et la version alpha est téléchargeable depuis le 23 Mars sur la page projet d’Escaryper.
    • Il suffit d’un smartphone sous Android pour y jouer mais malheureusement pour les utilisateurs d’iPhone, la version sous iOS n’est pas encore disponible donc n’hésitez pas à nous suivre sur les réseaux sociaux pour être à l’affût d’une annonce !
